# blog

Blog using Vercel and Next.js

## Requirements

- [Vercel CLI](https://vercel.com/download)
- [Node.js](https://nodejs.org/en/) v12 or later
- [Yarn](https://yarnpkg.com/getting-started/install)

## Install

```bash
yarn install
```

## Development

```bash
vercel dev

# or

vc dev
```

## Deploy

```bash
vercel

# or

vc
```
